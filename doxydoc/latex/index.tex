 \doxysection*{R-\/\+TYPE\+: A Modern Recreation}

Welcome to {\bfseries{R-\/\+TYPE}}, a modern reimagining of the iconic side-\/scrolling shoot \textquotesingle{}em up game. Immerse yourself in this multiplayer network game where players band together to fend off waves of enemies in a fast-\/paced, action-\/packed space adventure.

{\bfseries{R-\/\+TYPE}} combines classic gameplay mechanics with modern networking features, allowing players to connect across the globe and enjoy intense battles in space.

\DoxyHorRuler{0}


\doxysubsection*{Game Overview}

{\bfseries{R-\/\+TYPE}} is a horizontal shoot \textquotesingle{}em up game set in space, where you and other players battle alien forces. Equipped with powerful weapons, upgrades, and boosts, your mission is to survive and defeat as many enemies as possible while navigating through dangerous space environments.

\DoxyHorRuler{0}


\doxysubsection*{R-\/\+TYPE Server\+: The Core of the Game}

The {\bfseries{R-\/\+TYPE Server}} is the heart of the game. It handles all network operations, player connections, game state management, and interactions between players and enemies.

\begin{quote}


{\bfseries{Written in C++}}

\end{quote}


\doxysubsubsection*{Usage}


\begin{DoxyPre}{\ttfamily USAGE: ./r-\/type\_server -\/p port
    port        the port number
}\end{DoxyPre}


Once launched, the server manages the game state and allows multiple GUI clients to connect and interact with it.

\DoxyHorRuler{0}


\doxysubsection*{GUI Client\+: Your Window into the Game}

The {\bfseries{GUI Client}} allows players to interact with the game visually, controlling their spaceship, shooting enemies, and seeing other players.

\doxysubsubsection*{Usage}


\begin{DoxyPre}{\ttfamily USAGE: ./r-\/type\_gui -\/p port -\/n name [-\/h host]
    port        the port number
    name        the player name
    host        the server host (default: localhost)
}\end{DoxyPre}


The GUI displays menus, game levels, and other players in real-\/time.

\doxysubsubsection*{Client-\/\+Server Communication Protocol}

The {\bfseries{GUI}} communicates with the server using a well-\/defined protocol. For more details, see the \href{./server/readme.md}{\texttt{ server protocol documentation}}.
